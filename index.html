TheZZAZZGlitch's Glitch Research Archives

An archive of all of the things I did for my silly Pokémon glitch research YouTube channel.

  • abc-movies - BizHawk movies for my attempts at the Pokémon Gen I A Button Challenge.
    • PokemonBlue69xA.bkm - 69xA, my original attempt
    • DittoStrats59xA.bkm - 59xA with Ditto Strats
  • ace-listings - Code listings for my old Gen I Arbitrary Code Execution demonstrations.
    • 8F_CreepypastaASM.asm - source for the creepypasta save file
    • 8F_PongASM.asm - source for miniature pong (more like breakout with no blocks)
  • ace-savefiles - Save files for my old Gen I Arbitrary Code Execution demonstrations.
    • CreepypastaSaveFile.SAV - the creepypasta save file
  • crashsound-dumper - Toolchain for dumping Pokémon Emerald from its crash sound.
    • https://github.com/zzazzdzz/zzazzdzz.github.io/tree/master/crashsound-dumper
  • dpp-bulbasaur - The "definitely legit Bulbasaur" used for my Gen IV glitch move demonstrations.
    • DefinitelyLegitBulbasaur.sav - the Bulbasaur save file in question
  • emer-overlong-names - Collection of scripts to extract information about overlong move/Pokémon names in Emerald.
    • EmeraldDecamarkNames.zip - script to extract information about Decamark names
    • EmeraldGlitchMoveNames.zip - script to extract information about glitch move names
  • emu - Informational page about emulation accuracy, mostly intended for my April Fools events.
  • fools2014-mirror - Mirror of April Fools 2014
    • Togepi8FSave2.SAV - the April Fools 2014 save
  • fools2016-mirror - Mirror of April Fools 2016
    • missingno.exe.zip - the April Fools 2016 game
  • fools2024-mirror - Mirror of April Fools 2024
    • https://www.dropbox.com/scl/fi/fuz0ni5ybp67xa3g05thh/fools2024.zip?rlkey=5s53nrzhqglamux6c4twat9z1&st=e7z5dqtb&dl=0
  • gcl-thread-backups - PDF versions of my early threads on Glitch City Laboratories
    • 8FArbitraryCodeExecution_Backup.pdf - "Arbitrary Code Execution in Red/Blue using the 8F item"
    • EnglishRedBlueYellowDokokashira_Backup.pdf - "English R/B/Y equivalent of Dokokashira door glitch"
    • SuperGlitchResearch_Backup.pdf - "Making Super Glitch moves useful"
  • gsc-coincase-anal - Lua script for statically analysing code paths executed with the Coin Case, useful for finding non-ACE Coin Case glitches ("anal" stands for analysis)
    • coincase_memcorruption_checker.lua - Lua script in question
  • gsc-coincase-credits - First ever TAS attempt utilizing the Coin Case glitch for credits warp (was rejected, quite rightfully)
    • tzg_gold_coincase.vbm - VBA movie file
  • gsc-rce - helper scripts and PoC binaries for Remote Code Execution in Pokémon Crystal.
    • https://github.com/zzazzdzz/zzazzdzz.github.io/tree/master/gsc-rce
  • lua-techno - Windows reproduction of the TECHNO DOS virus, used as a demonstration for unsandboxed Lua scripting in emulators.
    • techno.exe.zip - raw zipped EXE file
    • techno.lua - same EXE in a self-contained Lua script
  • oob-diagrams - Diagrams and maps for out of bounds geography in Gen I games
    • OOB_Diagram.png - general OOB diagram
    • OOB_PalletTown.png - full-scale out-of-bounds map of Pallet Town
    • OOB_SeaRoute20.png - full-scale out-of-bounds map of Sea Route 20
  • rby-exp-groups - Documentation on glitch experience group formulas in R/B/Y
    • GlitchExperienceGroups.png - table of glitch experience groups
  • rby-trainer-lists - Documentation on all trainer rosters (including glitch trainers and glitch rosters) in Gen I
    • FullTrainerListG.txt - list of all trainers in Pokémon Green
    • FullTrainerListRB.txt - list of all trainers in Pokémon Red/Blue
    • FullTrainerListY.txt - list of all trainers in Pokémon Yellow
  • rse-ace-save - A save file
    • PokemonEmeraldArbitraryCode.sav - ACE save file for Emerald
    • PokemonEmeraldArbitraryCodeSavestate.SGM - VBA save state for the above ACE save file
  • sram-corruption-sim - Toolchain for programatically saving and reproducing SRAM corruptions of glitch Pokémon sprites
    • https://github.com/zzazzdzz/zzazzdzz.github.io/tree/master/sram-corruption-sim
  • translation-hacks - Translation hacks of Pokémon games (translation are intentionally very bad)
    • BingCrys.gbc - Pokémon Crystal translated with Bing
    • BlueTranslate.gbc - Pokémon Blue translated with Google Translate
    • BingEmeraldToolset.zip - an unfinished toolchain for creating a Pokémon Emerald Bing translation
  • truecrycoins - Automated script which attempts to reproduce 4 4's true cry, a rare unexplained glitch effect (which was likely an emulation error or some memory hack i forgot about). Lovingly named "Truecrycoin miner".
    • truecrycoins.zip - The truecrycoin miner
  • vba-exploits - ACE/infoleak exploits for the Visual Boy Advance emulator.
    • VBA18_CodeExec.zip - VisualBoyAdvance 1.8.0 - stack-based buffer overflow in XPC file parsing results in code execution; writeups and PoC files
    • vba_elfexploit.zip - VisualBoyAdvance & VisualBoyAdvance-M - vulnerabilities in ELF file parser allow for code execution and information disclosure; writeups and PoC files
  • yellow-debug-rom - Pokémon Yellow Debug ROM
    • DebugYellow.gbc - Debug Yellow ROM
